# Reassign Articles App

## Description:

This app allows agents to change the authorship of existing Help Center articles. This can be useful if an agent leaves your Zendesk.

## App location:

* User sidebar

## Features:

* Prevents reassignment to non-agents

## Set-up/installation instructions:

No special installation or setup is required.
